## Description

**DevDiggers Framework** is a powerful plugin that prevents unauthorized access, ensures a seamless user experience, and protects your valuable customer data and transactions. Safeguard your online store from malicious bots and spam with its advanced captcha functionalities.

## Features
```
* Supports Google reCAPTCHA, Cloudflare Turnstile CAPTCHA and hCaptcha.
* Supports Google reCAPTCHA v2 checkbox, v2 invisible and v3.
* Choose Google reCAPTCHA v2 checkbox size either as normal or compact.
* Choose hCaptcha size either as normal or compact.
* Module can be enabled/disabled by the admin.
* Supports different language selection for the CAPTCHA.
* Auto detection of the WordPress language for the CAPTCHA.
* Set error message to display users in case of not verifying the CAPTCHA successfully.
* Shortcode is provided for page builders.
* Whitelist IP addresses to not show the CAPTCHA.
* Exclude user roles to verify the CAPTCHA.
* Choose the CAPTCHA theme either as light or dark.
* Disable submit button until the CAPTCHA is verified.
* Add CAPTCHA on WordPress login, registration, lost/reset password and post comment forms.
* Add CAPTCHA on WooCommerce login, registration, lost/reset password, all users checkout, guest checkout, add payment method, pay for order, review product, account details, billing address and shipping address forms.
* Choose CAPTCHA position for WooCommerce checkout form.
* Enable CAPTCHA for Contact form 7 forms.
* Add CAPTCHA on WooCommerce Affiliates registration, become an affiliate and settings forms.
* Add CAPTCHA on MultiPOS - Point of Sale for WooCommerce login form.
* Multilingual is supported with the given .pot file in the plugin.
* And much more...
```

## Installation

1. Upload the `devdiggers-framework` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Configure the plugin using the `Points and Rewards` menu.
